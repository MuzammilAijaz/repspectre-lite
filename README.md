# Overarching Purpose
- Detect lifts using maching learning

# Project Goal
- be able to detect when the user has started a lift.
- be able to distinguish between lifts and simple act barbell picking, dropping, movement, rolling.
- be able to start collecting acceleration/other data when a lift has been detected.
- be able to identify the acceleration of the lift.
- be able to send data via ble to mobile.

- Optional :
    - be able to apply DSP techniques to reduce noise in acceleration data.
